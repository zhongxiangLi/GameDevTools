# GameDevTools
游戏开发日常用到的工具集

将GameDevTools.bat 创建快捷方式 放到桌面就可以快速打开

网络慢的可以尝试下载Release版本

https://github.com/ThisisGame/GameDevTools/releases

# 搜索工具

比如我想查找打图集工具，关键词 texturepacker。
那么输入 texture 或者 pack 或者 tex ，只要输入关键词部分字母 即可查找。

输入工具序号，即可使用工具

tips:
在 search 界面，输入 : 再输入系统命令，可以直接执行系统命令。

例如:
在search 界面输入

```
:ping www.baidu.com
```

即可执行 ping 百度的命令。
